PickMeal 프로젝트

게임 요소를 결합한 맞춤형 식사 추천 & 커뮤니티 플랫폼

주요 기능
- Spring Security 기반 인증/인가 체계 구축 (Security,OAuth2,JWT)
- 커뮤니티 비즈니스 로직 및 데이터 관리 (CRUD, MyBatis)
- 확률형 게임 콘텐츠 및 알고리즘 설계 (알고리즘, 스케줄링)
- 클라우드 기반 미디어 관리 및 시스템 자동화 (AWS S3, Scheduling)
- 실시간 주변 맛집 시각화 및 공유 기능(Kakao Map API)
